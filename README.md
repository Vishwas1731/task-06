Project Overview

This project analyzes monthly sales trends by calculating total revenue and order volume from an online sales dataset using SQL aggregation functions. The goal is to understand business performance over time and identify peak sales periods.

- Objective

Analyze monthly revenue using SUM()

Calculate order volume using COUNT(DISTINCT order_id)

Group data by month and year using EXTRACT()

Sort results using ORDER BY

Limit results for specific time periods using LIMIT

- Tools & Technologies

Database: / MySQL / 

Language: SQL

Dataset: Online Sales (orders table)

 Dataset Description

Table Name: orders

Column Name	Description
order_id	Unique order identifier
order_date	Date of purchase
amount	Revenue per order
product_id	Product identifier

-Results

Monthly revenue and order volume were successfully calculated.

Data was grouped by year and month for trend analysis.

Results were sorted to identify top-performing months.

Limited results helped focus on specific time periods.
